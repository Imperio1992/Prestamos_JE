# Prestamos_JE
Préstamos JE es un sistema web desarrollado en PHP y MySQL para gestionar préstamos diarios. Permite registrar clientes, asignar préstamos a asesores, llevar control de cobranza diaria y administrar caja (efectivo y banco). Incluye un sistema de roles (GERENTE y ASESOR) con seguridad por sesión para controlar accesos y proteger la información.
